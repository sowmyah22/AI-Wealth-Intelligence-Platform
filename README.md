# AI-Wealth-Intelligence-Platform

## About the Application
An AI-powered investment intelligence platform that combines portfolio analytics,
Retrieval-Augmented Generation (RAG), agentic AI, market intelligence, and
recommendation engines to generate personalized portfolio insights.

The platform analyzes user portfolios, identifies risks, retrieves historical
portfolio context, and generates AI-powered investment briefings with explainable
recommendations.
